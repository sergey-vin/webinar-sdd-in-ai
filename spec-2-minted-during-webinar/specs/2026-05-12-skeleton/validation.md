# Фаза 1: Скелет — Validation

## Критерии приёмки

### Auth
- [ ] Можно зарегистрироваться по email+password
- [ ] Можно залогиниться как user_a (user@cuthere.test)
- [ ] Можно залогиниться как user_b (premium@cuthere.test)
- [ ] Незалогиненный пользователь редиректится на страницу логина
- [ ] Кнопка "Выйти" в профиле работает

### Навигация
- [ ] Bottom tabs видны на мобильном (< 768px): Поиск, Мои маршруты, Цены, Профиль
- [ ] Top navbar виден на десктопе (>= 768px)
- [ ] Все 4 таба кликабельны и ведут на соответствующие страницы
- [ ] Страницы-заглушки отображают стилизованное сообщение "скоро будет"

### Профиль
- [ ] Отображаются: имя, email, аватар (placeholder если нет)
- [ ] Работает переключение языка ru/en — интерфейс меняется
- [ ] Предпочтения транспорта: чекбоксы сохраняются в БД
- [ ] Настройки уведомлений: toggle сохраняется в БД
- [ ] Переключатель тестовых юзеров: при клике — перелогин под другого юзера
- [ ] Роль юзера видна (user / premium бейдж)

### БД
- [ ] Таблицы существуют: profiles, cities, transport_types, routes, route_segments
- [ ] RLS: пользователь видит/редактирует только свой profile
- [ ] RLS: cities, transport_types, routes, route_segments доступны на чтение всем авторизованным
- [ ] Seed: оба тестовых юзера созданы с корректными ролями

### i18n
- [ ] Приложение запускается на русском по умолчанию
- [ ] Переключение на английский работает из профиля
- [ ] Все видимые строки (навигация, профиль, заглушки) переведены на оба языка

### PWA
- [ ] manifest.json присутствует и корректен
- [ ] Service worker регистрируется
- [ ] Приложение можно "установить" на мобильном (Add to Home Screen)

### Визуальный стиль
- [ ] Акцентный цвет — коралловый (#FF6B6B)
- [ ] Скруглённые углы на карточках и кнопках
- [ ] Шрифт Inter или Nunito
- [ ] Mobile-first: на 320px всё читаемо, ничего не обрезано

### Playwright
- [ ] Тест: логин user_a → переход по 4 табам → всё отображается
- [ ] Тест: в профиле переключение на user_b → роль меняется на premium
- [ ] Тест: переключение языка ru → en → тексты меняются
- [ ] Все тесты проходят в CI (или локально)
